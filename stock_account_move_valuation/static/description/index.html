Stock Accounting Move Valuation

This module restores stock valuation journal entry creation when a stock picking is validated, similar to the behavior users were familiar with in Odoo 18.

What it does

  • Creates account moves for done stock moves on real-time valued product categories.
  • Adds a category option to enable journal entry creation on move validation.
  • Lets you select the stock journal directly from the product category.

Configuration

  1. Open a product category configured with real-time inventory valuation.
  2. Enable Create Journal on Move.
  3. Select the journal to use for the generated valuation entries.
  4. Validate incoming or outgoing pickings for products in that category.

Use case

It is useful when you want stock valuation postings to happen immediately at picking validation and keep a workflow closer to the previous version's accounting behavior.
